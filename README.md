# 📖 Overview

This project analyzes insurance policy and customer data to identify business trends, evaluate claim performance, monitor premiums and coverage, and support data-driven decision-making.

Using **Python for data cleaning and EDA**, **SQL for business analysis**, and **Power BI for visualization**, this project transforms raw insurance data into actionable business insights.

---

# 💼 Business Problem

Insurance companies manage thousands of customer policies and claims every day. Manually analyzing this large volume of data makes it difficult to:

- Monitor premium collection efficiently.
- Track claim performance.
- Identify customer trends.
- Evaluate policy performance.
- Support business decisions using data.

This project provides an interactive analytics solution that enables stakeholders to monitor business performance using KPIs and dashboards.

# 📌 Business Questions & Key Insights

### 1. Which policy type generates the highest premium?

Health insurance policies contribute the highest premium revenue.

### 2. Which customer segment purchases the most policies?

Adult customers represent the largest customer segment.

### 3. What is the claim settlement performance?

Most claims are successfully settled, while a smaller percentage remain pending or rejected.

### 4. How do active and inactive policies compare?

Active policies significantly outnumber inactive policies.

### 5. Which policy categories contribute the highest coverage?

Coverage amounts vary significantly across policy categories, highlighting differences in customer protection levels.

# ▶ How to Run This Project

### 1. Clone the repository

```bash
git clone https://github.com/Arpit9516/Insurance-Data-Analytics.git
```

### 2. Install dependencies

```bash
pip install pandas numpy matplotlib seaborn
```

### 3. Open the Jupyter Notebook

Run the notebook inside the `notebooks` folder.

### 4. Execute SQL Scripts

Run the SQL queries available in the `sql` folder.

### 5. Open Power BI Dashboard

Open the `.pbix` file inside the `powerbi` folder.
